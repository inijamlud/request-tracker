# Request Tracker

A fullstack request management app built with Next.js, Prisma, and PostgreSQL.

## Features

- Create, edit, delete requests
- Status flow: PENDING → SUBMITTED → DONE
- Priority & due date
- Tags with multi-filter
- Dashboard with charts
- Activity feed & timeline
- Realtime updates (SSE)
- Dark mode

## Tech Stack

- Next.js 15 (App Router)
- Prisma ORM + PostgreSQL
- TailwindCSS v4
- Recharts
- Zod

## Getting Started

```bash
npm run dev
```
